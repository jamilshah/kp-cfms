About Department-Level Access Control:

Control whether users can only access data from their assigned department, or if they can see all departments within your organization.

  • Enabled: Users (DA, Accountant) see only their department's bills, budgets, etc.
  • Disabled: All users can see all departments (current default behavior)
  • Note: TMO, Finance Officers, and TMA Admins always see all departments
