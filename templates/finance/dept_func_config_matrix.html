About Fund Configuration

This configuration is fund-agnostic. You define which NAM heads are allowed for each department-function combination, and this configuration applies to all funds (GEN, DEV, etc.).

To create actual budget heads:

  1. Configure allowed NAM heads here (this page)
  2. Click "Bulk Create Budget Heads" button above and select fund (GEN, DEV, etc.)
  3. Or run command line: python manage.py create_budget_heads_from_config --fund GEN

Example: If you configure "Admin + Municipal Administration" with 98 NAM heads, creating budget heads will generate 98 budget heads for the selected fund, all ready to use (posting_allowed=True).
